# Flight Information App

An Android flight information application using Skyscanner's Backpack UI components.

## Features
- Flight number display
- Departure and arrival airport codes
- Departure and arrival times
- Modern Backpack UI design

## Technologies Used
- Kotlin
- Android SDK API 33
- Backpack Android Library 43.0.0
- Material Design Components

## Setup Instructions
1. Clone this repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator or physical device
